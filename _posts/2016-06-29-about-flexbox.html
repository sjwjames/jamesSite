The CSS3 Flexible Box, or flexbox, is a layout mode providing for the arrangement of elements on a page such that the elements behave predictably when the page layout must accommodate different screen sizes and different display devices. For many applications, the flexible box model provides an improvement over the block model in that it does not use floats, nor do the flex container's margins collapse with the margins of its contents.

FlexBox也称为css弹性布局盒子，是css3中定义的一种新的布局模式。主要用途是可以在子元素高度宽度未知或者是动态的时候自动去调整其子元素的宽度或者高度使其能在不同分辨率的设备下用比较优美的方式去填充布局。flexbox通常用于小的应用组件中， Grid布局模式则应用于大规模的布局中。

flex盒子模型由容器（flex container）以及容器的直接子元素（flex item）组成。 以容器（flex container）定义，有主轴（main axis）以及纵轴（cross axis），主轴为默认横向，默认从左到右，纵轴默认为纵向，默认从上到下（当布局排列方式变化的时候，main axis跟cross axis的方向也会跟着变化

下面介绍各个属性及其不同值对应的布局情况。

flex-direction

flex-direction一共有四个值，row|row-reverse|column|column-reverse.row/row-reverse是按照行来排列，只是开始顺序不一样。column同理

See the Pen flex-direction by James (@sjwjames) on CodePen.

flex-wrap

flex-wrap主要控制的是子元素的换行规则，默认no-wrap，不换行，所以可以看到即使给子元素设置了宽度实际宽度也会变化;wrap则是换行，可以同时设置flex-direction来设置新行的排列规则;wrap-reverse则是反向换行。此处需要注意的是当 flex-direction是column/column-reverse的时候,设置flex-wrap为no-wrap是没用的。

See the Pen flex-wrap by James (@sjwjames) on CodePen.

flex-flow

flex-flow是flex-direction和flex-wrap的简要写法

justify-content

justify-content控制子元素在flex中的排布规则，flex-start和flex-end是从盒子的头尾开始排布，center则是居中排布，space-around会在每个元素周围排布相同的间距，并且两边到边框的距离相等，space-between的话与space-around相同，只是两边到边框没有距离

See the Pen JKyPEy by James (@sjwjames) on CodePen.

align-items

align-items控制子元素在同一行时在纵轴的排列规则,取值有flex-start,flex-end,center,stretch,baseline。

align-content

align-content的值域与justify-content一样，控制多行的box中行与行之间的排列规则，规则与取值之间的对应类似justify-content。
以上都是container的属性，下面列举一些item的属性。

order

顾名思义，控制item在container中的顺序

flex-grow

设置item的权重，根据权重比来分配各个item的空间，default值为0

flex

flex-grow flex-shrink flex-basis的合写

align-self

这个属性控制了item的排列规则，可以覆盖container已经设置的align-items值
